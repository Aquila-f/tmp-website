## Overview

**Agent-Brain** is a 30-day technical blog series that documents the complete process of designing and implementing an LLM agent workflow from scratch. This project was recognized with an **Honorable Mention Award** at the 2025 iThome Ironman Technical Writing Challenge.

## Key Highlights

- Built a modular LLM agent system from scratch, implementing finite-state reasoning stage with tools integration
- Authored a technical blog series detailing design principles for scalable LLM-based agent workflows
